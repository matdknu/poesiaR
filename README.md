# poesiaR
Proyecto de poesía en lengua castellana en R.

Se tiene como objetivo descargar los poemas que se encuentran la página https://www.poemas-del-alma.com. Se han cubierto autores como Pablo Neruda, Lope de Vega, Jorge Luis Borge, Ida Vitale, entre otros

## Posible análisis

Procesar datos de texto se ha vuelto de vital importancia para las ciencias sociales. Los análisis de "big data" puede sumarse a los análisis interpretativos de la teoría literaria, contexto del autor, entre otros. 

La riqueza de esta base de datos se sustena en los más de mil poemas que existe en una base de datos de tipo estructurada y fácil manejo. Sin embargo, tiene limitancies tales como:

1. No están todos los poemas de los autores, solo una selección que la página ha realizado.
2. Los poemas no tienen los años correspondientes, aunque buscará encontrar una solución a ello mediante una codificación manual o mediante webscrapping-
